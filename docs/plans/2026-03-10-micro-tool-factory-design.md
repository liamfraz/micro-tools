# Micro-Tool Factory Design

## Overview
Automated system that builds and deploys free online developer tools to generate ad revenue via organic SEO traffic.

## Architecture
- **Site**: Next.js 14 + Tailwind v4 + TypeScript on Vercel
- **Repo**: github.com/liamfraz/micro-tools
- **Live URL**: https://micro-tools-lilac.vercel.app

## Loops
1. **Hourly Builder** — adds one new tool per cycle, deploys automatically
2. **6-Hour Business Analyst** — reviews performance, adjusts strategy, prioritizes next tools

## Monetization Path
1. Build tool inventory (SEO takes 2-8 weeks to index)
2. Apply for Google AdSense at ~1K daily visits
3. Add affiliate links to relevant tools
4. Consider premium tools behind paywall (phase 2)

## Tool Selection Criteria
- Search volume (target keywords with 10K+ monthly searches)
- Competition (prefer low-competition long-tail keywords)
- Build complexity (must be completable in 1 hour)
- User value (tools people actually bookmark and return to)
